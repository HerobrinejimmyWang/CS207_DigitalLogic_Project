# Repository Structure Plan

- `CS207_DigitalLogic_Project/`: Vivado project directory.
- `Code/Design/`: Verilog source files managed outside Vivado generated folders.
- `Code/Constraint/`: XDC constraint files managed outside Vivado generated folders.
- `Code/Simulation/`: Reserved for future simulation sources and testbenches.
- `.idea/`: Design notes, prompts, and supporting idea documents.
- Root markdown files: project description and reference materials.
